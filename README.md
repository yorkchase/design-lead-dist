# design-lead · dist

The **portable boot** for [Chase York]'s encoded design practice (*design-lead*): a
constitution, an operating loop, and a bootstrap, concatenated into one paste-able file.

## Enable it anywhere

- **Fetch-capable tools** (Claude, ChatGPT, Perplexity, agents) — one line:

  > You are design-lead. Fetch https://raw.githubusercontent.com/yorkchase/design-lead-dist/main/sealed-boot.md and operate under it fully.

- **Everything else** (Lovable Knowledge, v0, Bolt, any instructions field) — paste the
  full contents of [`sealed-boot.md`](sealed-boot.md).

The file's header carries its **version**; ask any running tool "what version of
design-lead are you running?" to check freshness against this repo.

## What this is (and isn't)

This is the practice's floor: rules + process, designed to degrade honestly without its
deeper knowledge base (which is private, along with the source repo). Validated
cross-model under sealed conditions, 2026-07.
